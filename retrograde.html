About

Retrograde was created as part of an individual project for IT1552 Web Development module (attained Distinction grade).

This was my first ever web development project, where I was pushed to explore just how much I could learn and achieve with just a few months of experience in HTML, CSS and JavaScript.

Games

With references to open source codes, I was able to recreate the Snake game. Afterwards, I figured out how to add more game features such as the recording of personal best score, community leaderboard, changing of snake skin and having different game modes.

Later on, I created the Puzzle game while experimenting with HTML5's drag and drop API.

User Accounts

Using local and session storage, I created the following account features for the users:

  • Sign Up
  • Log In
  • Changing of username, password or profile picture
  • Using/Sharing of referral code
  • View user profile/game statistics
