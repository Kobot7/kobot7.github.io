About

Lorem Ipsum Forum was created as part of a group project for IT2555 Applications Security Project module (attained Distinction grade).

For this project, our team developed a vulnerable and secure version of an online forum named “Lorem Ipsum”.

Lorem Ipsum is an online forum that allows users to communicate with each other on a variety of subjects. They will be able to read all the threads on a particular subject or discussion, as well as contribute their own posts. There is a need for administrators to vet through messages and take appropriate action to supervise the community as well.

The vulnerable version of Lorem Ipsum contains the implementations of the 6 out of the Top 10 OWASP Web Vulnerabilities that were chosen by our team. The team members as well as our chosen vulnerabilities are as follows:

  • Ko Jia Ling (Myself) – Broken Access Control, Injection
  • Sarah Bagarib – Broken Authentication, Security Misconfiguration
  • Muhammad Ennaayattulla – Sensitive Data Exposure, Cross Site Scripting

The secure version of Lorem Ipsum is mitigated against the 6 chosen vulnerabilities by the respective member in charge. Additional mitigation against a few of the other vulnerabilities found during the analysis of the vulnerable version of Lorem Ipsum has been implemented as well.

Base Features

Although the main focus of this project was the securing of web vulnerabilites, our team had to create a functional forum first. The following are features of the website done by me:

  • Home Page
  • Search Bar – Filter posts
  • Adding Posts
  • Commenting on posts
  • Replying to comments
  • Upvote/Downvotes for posts and comments
  • Sending feedback
  • Sign Up – Creating new account
  • Log In
Securing of Web Vulnerabilites

The 2 OWASP Top 10 Web Vulnerabilities I chose were broken access control and injection. In addition, I implemented a monitoring and logging system for the secure version of Lorem Ipsum. The following are security features that I implemented:

  • Validate hidden form field values
  • Retrict access to admin pages
  • Prevent SQL injection
  • Implemented logging system
  • Implemeted Flask Monotoring Dashboard
  • Created monitoring dashboard for errors and activities
