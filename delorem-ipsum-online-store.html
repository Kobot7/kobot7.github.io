About

Delorem Ipsum Online Store was created as part of a group project for IT1566 - App Development Project module.

Delorem Ipsum is a web-based Python application that serves as an online pharmacy store.

I was mainly in charge of the admin interface for this project. The features completed by me are as follows:

  • Admin dashboard
    • Items with low stock
    • Pending transactions
    • Best selling/Most viewed items
  • Product catalog
    • Add new product
    • Edit existing product
    • Add stock
    • Export product information (excel sheet)
  • Product statistics
    • Highlight products below/almost reaching threshold stock value
    • Graphical view of product statistics
    • Export product statistics (excel sheet)
  • Transactions
    • View transaction infromation
    • Mark transactions as complete/incomplete
    • Export transaction information (excel sheet)
